Replace all dashboard charts with Chart.js line charts

Dependencies:
- Add chart.js ^4.4.4
- Add react-chartjs-2 ^5.2.0

New component:
- src/components/dashboard/LineChart.tsx: reusable line chart wrapper with:
  - Configurable series (multi-line support: Pemasukan + Pengeluaran, etc.)
  - Color themes: emerald (Pemasukan/Trafik), amber (Pengeluaran), blue, rose
  - Customizable yFormatter (auto-format IDR with thousand separator)
  - Responsive, smooth bezier curves (tension 0.4)
  - Dark tooltip + clean legend

Charts replaced (3 total):

1. /dashboard (Overview) - "Trafik 7 Hari":
   Before: sparkline bars (custom divs, height: 80/100px each)
   After: LineChart single series "Page Views", labels formatted as Indonesian weekday + day

2. /dashboard/keuangan - "Tren Keuangan 6 Bulan":
   Before: stacked bars (custom divs, manual height calculation)
   After: LineChart dual series with custom yFormatter (IDR with separator):
   - Series 1: Pemasukan (emerald)
   - Series 2: Pengeluaran (amber)
   - Labels: Indonesian month name + 2-digit year (e.g., "Jan '26")

3. /dashboard/trafik - "Trafik 7 Hari Terakhir":
   Before: custom bars (h-32 each)
   After: LineChart single series "Page Views"

Improvements over custom SVG/divs:
- Tooltip with value on hover
- Smooth line curves (tension 0.4)
- Multi-series with shared tooltip
- Consistent typography & spacing
- Responsive auto-resize
- Loading state handled by Chart.js animations
- Color-coded legend in top-right
